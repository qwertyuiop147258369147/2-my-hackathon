# 🍏 智慧健康日誌與動態風險評估系統 (Smart Health Log & Risk Assessment System)

本專案是一款專為個人健康管理設計的輕量化 Dashboard 系統。結合直觀的前端互動介面與動態評估演算法，幫助使用者即時追蹤健康軌跡，並透過決策樹模型進行動態健康風險評估。

## 🚀 核心功能與技術亮點

- **90天大數據動態軌跡**：系統自動模擬並渲染長達 90 天的真實健康波動軌跡，包含睡眠時數、當日步數與心情分數。
- **🧠 決策樹動態評估演算法 (Dynamic Decision Tree)**：
  - 前端即時根據使用者輸入的生理數據（如：睡眠不足 5 小時且步數低於 4000 步）進行多分支條件決策。
  - 自動判定並歸類為 `高風險`、`中風險` 或 `低風險`。
- **即時響應式響應 (Reactive Dashboard)**：當使用者送出今日新日誌時，系統採不刷新頁面的異步處理技術，秒級將全新數據與動態評估標籤插入歷史日誌頂端。
- **高質感極簡 UI 視覺設計**：仿 Apple 風格的圓角卡片排版、動態滾動條歷史列表，並對不同風險等級進行高對比度色彩標籤（Badge）視覺優化。

## 🛠️ 技術棧 (Tech Stack)

- **前端 (Frontend)**：HTML5, CSS3 (Flexbox/Grid 響應式佈局), JavaScript (ES6+, Async/Await)
- **後端架構相容性**：原生支援 Java HttpServer / RESTful API 整合架構。

## 📊 決策評估邏輯說明

系統內建的決策樹評估核心遵循以下規則：
1. **高風險 (High Risk)**：睡眠時數 < 5.0 小時 且 當日步數 < 4000 步。
2. **中風險 (Medium Risk)**：睡眠時數 < 6.0 小時 或 心情分數 < 5 分 或 當日步數 < 5000 步。
3. **低風險 (Low Risk)**：其餘正常健康狀態。
